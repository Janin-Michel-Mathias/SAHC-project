# Décision sur le système d'email
Date: 30-03-2026
## Problème
Comment gérer l'envoi de mail dans notre application dans le temps imparti pour le projet

## Solutions envisagées
- Mettre en place notre propre serveur SMTP
- Mettre en place la configuration nécéssaire pour utiliser un serveur SMTP existant (example google)
- Utiliser une API externe proposant de gérer l'envoi de mail

## Solution gardée
Utiliser une API externe proposant de gérer l'envoi de mail

## Raison
A cause du temps limité pour ce projet nous ne disposons pas du temps nécéssaire pour les autres options.
