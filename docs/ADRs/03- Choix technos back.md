# Choix des technos à utiliser sur le backend
Date: 30-03-2026
## Problème
Quelle stack technique choisir pour le backend de notre application

## Solutions envisagées
- NestJS
- Prisma
- Go

## Solution gardée
NestJS

## Raison
Etant donné le temps limité pour ce projet, et malgré les meilleures performances de Go, il a été décidé de rester sur le framework le mieux maitrisé par l'équipe, donc NestJS.
