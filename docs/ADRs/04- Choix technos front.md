Date: 30-03-2026
## Problème

## Solutions envisagées
- Next

## Solution gardée

## Raison
Etant donné les compétences de l'équipe actuelle, il a été décidé de rester sur les technos déjà maitrisées par l'équipe, Next permettant de faciliter le développement d'une application React, il a donc été choisi.
