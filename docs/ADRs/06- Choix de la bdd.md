# Choix de la base de données
Date: 30-03-2026
## Problème
Quelle base de donnée utiliser pour notre application

## Solutions envisagées
- MariaDB
- Mysql
- Oracle
- Postgres
- SQLite

## Solution gardée
Postgres

## Raison
Etant donné les habitudes de l'équipe nous avons décidé d'utiliser Postgres. SQLite aurait pu être une bonne option mais sans savoir la taille que l'historique va atteindre, nous avons choisi la sécurité.
